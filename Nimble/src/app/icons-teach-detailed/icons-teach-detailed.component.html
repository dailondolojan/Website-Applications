Types of Menus

Menus provide navigation to other elements and features. This means some common functionality may be hidden in a menu, especially on mobile devices where screen "real-estate" or screen space is limited.

Menu Icon Name Description
Hamburger Menu The three (or more) lines resemble a hamburger, hence the name. This type of menu usually hides the primary navigation (similar to a glossary). Often used on mobile, this icon can be clicked on to display the menu.
Overflow (or 3-dot) Menu The three dots (horizontal or vertical) resemble an ellipsis (...). This small menu holds items that are not displayed to reduce clutter on the screen or because there is no more space.
Drop-down Menu Upon clicking or hovering over this type of menu, a menu will drop down (hence the name) to display more options. It can often be clicked again to hide the menu.
