Complete List of Common Icons

Think about each icon's real-world counterpart. For example, "Search" uses a magnifying glass and "Attach" uses a paperclip.

Icon Label Note
Add Add new documents, messages, etc.
Attach Attach pictures or other files to a message
Back Return to the previous screen
Bookmark Mark a website or other item as a favorite
Brightness Control how bright your screen is
Chrome Internet Browser, Google Chrome
Close Exit or close a window.
Camera Take pictures
Delete Similar to "Close" icon
Delete Delete a character while typing
Trash Can also mean "Delete"
Download Save a file
Drop-down Displays a menu below
Compose Create an email or file
E-Mail Can also resemble postage stamps
Battery Fill amount indicates battery level
Firefox Internet browser, Mozilla Firefox
Gmail Common E-Mail service, Google Mail
Hamburger Menu Toggles display of a vertical menu bar
Internet Explorer Internet Browser, Internet Explorer
Zoom Out Make content smaller
Zoom In Make content larger
Mute Sound is off
Folder Holds a collection of files
Play Play song, video, etc.
Pause Stop a song, video, etc.
Power Power button to turn on/off a device
Print Print a file
Reply Respond to an e-mail
Reply All Respond to everyone in an e-mail
Safari Internet Browser, Safari
Save Save a file
Search Type in textbox next to "Search" icon
Send Send a message (icon is a paper airplane)
Settings Change preferences for your device
SMS Text Messaging
Star Same as bookmark/favorites
Camera Flip Switch between front and rear facing cameras
Overflow Menu (Or 3-dot menu) holds small menus
Toggle Switch Turn settings on/off
Voicemail Holds voice messages
WiFi Wireless internet
